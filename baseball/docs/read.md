# 개발 전 정리
- 1부터 9까지 서로 다른 수로 이루어진 3 자리의 수 맞추기
- 스트라이크, 볼, 낫싱
- 수를 비교해야겠군

# 기능 목록
- [x] 베이스볼 게임 App
- [x] 유저(플레이어)
- [x] 상대방(컴퓨터)
- [x] 숫자를 저장하는 녀석
- [x] 정답 숫자 생성기
- [x] 숫자를 비교해주는 클래스
- [x] 입력
  - 서로 다른 3자리의 수 입력
  - 게임이 끝난 경우 재시작 입력 (1 과 2)
- [x] 출력
  - 결과 출력
- [x] 예외
- 3자리 수 입력 예외
- 게임 재시작 입력 예외
